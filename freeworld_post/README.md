## 🚀 Quick Start Guide

### Prerequisites
- Flutter 3.10+
- Dart 2.19+
- Laravel 10
- PHP 8.1+

### Backend Installation
```bash
# Clone Repository
git clone https://github.com/yourusername/alu-free-media-backend.git

# Navigate & Install
cd alu-free-media-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

### Frontend Setup
```bash
# Clone Repository
git clone https://github.com/yourusername/alu-free-media-frontend.git

# Prepare Application
cd alu-free-media-frontend
flutter pub get
flutter run
```
